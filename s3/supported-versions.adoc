= サポートされるバージョン
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
StorageGRID でサポートしている S3 および HTTP のバージョンは次のとおりです。

|===
| 項目 | バージョン 


 a| 
S3 仕様
 a| 
_Simple Storage Service API Reference_2006-03-01



 a| 
HTTP
 a| 
1.1

HTTP の詳細については、 HTTP/1.1 （ RFC 7230~7235 ）を参照してください。

* 注： StorageGRID は、 HTTP/1.1 パイプラインをサポートしません。

|===

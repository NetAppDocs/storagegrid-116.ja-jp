= BMC インターフェイスの設定（ SG100 および SG1000 ）
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
サービスアプライアンスのベースボード管理コントローラ（ BMC ）のユーザインターフェイスには、ハードウェアに関するステータス情報が表示され、サービスアプライアンスで SNMP 設定やその他のオプションを設定できます。

= BMC インターフェイス（ SG6000 ）の設定
:icons: font
:imagesdir: ../media/


[role="lead"]
SG6000-CN コントローラのベースボード管理コントローラ（ BMC ）のユーザインターフェイスは、ハードウェアに関するステータス情報を提供し、 SG6000-CN コントローラの SNMP 設定やその他のオプションを設定できます。

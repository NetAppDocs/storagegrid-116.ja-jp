= SANtricity Storage Manager を設定します
:icons: font
:imagesdir: ../media/


[role="lead"]
SANtricity Storage Manager を使用して、 StorageGRID アプライアンスのストレージディスクおよびハードウェアコンポーネントのステータスを監視できます。このソフトウェアにアクセスするには、 E2700 コントローラ（アプライアンスのストレージコントローラ）の管理ポート 1 の IP アドレスが必要です。
